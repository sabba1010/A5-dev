Question-1: What are the different ways to select an element in the DOM?

Ans: The different ways to select an element in the DOM are:
1. GetElementById()
2. GetElementsByClassName()
3. GetElementsByTagName()
4. QuerySelector()
5. QuerySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans: The difference between innerHTML, innerText, and textContent is:
1. innerHTML: Processes HTML and is used to add or modify HTML content.
2. innerText: Retrieves only the visible text.
3. textContent: Displays hidden text as well.

Question-3: What is event delegation in the DOM?

Ans: The event delegation in the DOM is:
A technique in JavaScript where a parent element handles events for its child elements using event bubbling.

Question-4: What is event bubbling in the DOM?

Ans: The event bubbling in the DOM is:
It is a process where an event triggered on a child element bubbles up to its parent, grandparent, and eventually to the document.

Question-5: How do you create, add, and remove elements using JavaScript?

Ans:
Creating an Element: To create a new element, i can use document.createElement("tag"). This allows i create any HTML element dynamically in JavaScript. After creating the element, i can modify its properties like text, class, or id.
Adding an Element: To add the created element to the DOM, i can use methods like : appendChild(): Adds the element as the last child of a parent element. prepend(): Adds the element as the first child. insertBefore(): Adds the element before another existing element.
Removing an Element: To remove an element from the DOM, i can : removeChild(): Removes a child element from a parent element. remove(): Removes the element itself.
